# Machine Architecture Coursework 2019
(2019) Machine Architecture submodule coursework submission for the Computer Systems module at Durham University

## Description
Outputs an iterative sequence using Durham University's version of the [Little Man Computer](https://wikipedia.org/wiki/Little_man_computer)

## Questions
Outputs the sequence described iteratively given a starting point
### Basic
The basic sequence was given by:
